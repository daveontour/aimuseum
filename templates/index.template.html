        No Master Key Set

        A master key is required to use Sensitive & Private Data. Create one by entering a password below. Keep this password safe—you will need it to unlock records.

        Create New Master Key

        This action cannot be undone.

        Creating a new master key will delete all existing trusted keys. Any sensitive data encrypted with those keys will become permanently unrecoverable. Only the new master password will be able to access sensitive data going forward.

        Enter the new master password. This will be the only key that can access sensitive data.

        Create Trusted Key

        Create a new trusted key for a user password. The user password will unlock Sensitive & Private Data records. You need the master password to create keys.

        Delete Trusted Key

        Remove a trusted key. Enter the user password whose key you want to delete, and the master password. The master key cannot be deleted.

        Add Document Key

        Add a new keyring seat so a user can decrypt encrypted documents with that password. Requires the master password. The AI uses the same master key after you unlock it in this session.

        Remove Document Key

        Remove a keyring seat, revoking decryption access for that user. The master seat cannot be removed.

        Edit visitor hint

        Add hint for existing seat

        Choose a visitor keyring seat that has no hint row yet, then enter the text visitors will see on the unlock screen.
