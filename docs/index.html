  • HTTP API with middleware-based auth and request-scoped user context.

Security model

  • Session cookies, Argon2id passwords, encrypted sensitive blobs with keyring-derived keys.
  • Visitor keys and share links with granular feature flags.
  • Separate admin and launcher entry points where applicable.

May 2025 Project Origins

A learning activity that got out of control

I had been using AI tools, but just the normal chat bot type websites. Professionally, AI was coming up a lot more and I needed to know more of the tech in details, so since I learn best by doing, I started some of my own coding exercises to learn more about the tech.

One of the places I started was RAG (Retrieval-Augmented Generation) and I started to build my own RAG system. I started with archived WhatsApp conversations and Facebook messages and found the AI tools were able to answer questions about the conversations and provide insights into the conversations.

When I was coding up the system instructions, I playfully decided to give it a persoanlity and was amused and impressed by the results.
